# ChatBI股票助手 - 布林带异常点检测功能说明

## 📋 功能概述

stock_assistant-3.py 在 stock_assistant-2.py 的基础上，新增了 **布林带(Bollinger Bands)异常点检测功能**，可以检测股票的超买和超卖点。

---

## 🎯 新增功能

### boll_detection 工具

**功能描述：**
使用布林带模型（20日周期 + 2倍标准差）检测股票的超买和超卖异常点。

**参数说明：**
- `ts_code`（必填）：股票代码，如 '600519.SH'、'000858.SZ'
- `start_date`（可选）：开始日期，YYYYMMDD格式，默认为一年前
- `end_date`（可选）：结束日期，YYYYMMDD格式，默认为今天

**工作流程：**
1. 从本地SQLite数据库获取指定时间范围内的历史收盘价数据
2. 计算20日移动平均线（中轨）
3. 计算2倍标准差，确定上轨和下轨
4. 检测异常点：
   - **超买点**：收盘价突破上轨
   - **超卖点**：收盘价跌破下轨
5. 生成可视化图表（收盘价 + 布林带 + 异常点标记）

**返回内容：**
- Markdown格式的检测结果统计
- 异常点详情表格
- 可视化图表（布林带图）

## 🔧 技术细节

### 布林带计算公式

- **中轨（Middle Band）**: 20日移动平均线
  ```
  MB = MA(Close, 20)
  ```

- **上轨（Upper Band）**: 中轨 + 2倍标准差
  ```
  UB = MB + 2 × σ(Close, 20)
  ```

- **下轨（Lower Band）**: 中轨 - 2倍标准差
  ```
  LB = MB - 2 × σ(Close, 20)
  ```

### 异常点判定

- **超买点（Overbought）**: Close > Upper Band
  - 表示价格可能过高，存在回调风险
  
- **超卖点（Oversold）**: Close < Lower Band
  - 表示价格可能过低，存在反弹机会

### 数据处理

1. **数据获取**: 从SQLite数据库获取指定时间范围的收盘价
2. **指标计算**: 使用pandas的rolling窗口计算移动平均和标准差
3. **异常检测**: 比较收盘价与上下轨的关系
4. **可视化**: 绘制完整的布林带图和异常点标记

## 🎓 使用建议

### 适合的场景
- 识别股票的超买超卖状态
- 寻找潜在的买卖时机
- 技术分析辅助决策
- 教学演示和学习

### 不适合的场景
- 单一指标决策（应结合其他技术指标）
- 基本面分析替代
- 高频交易决策

### 最佳实践
1. **结合多个指标**: 布林带应与MACD、KDJ等指标配合使用
2. **观察趋势**: 布林带收窄表示波动减小，可能即将变盘
3. **关注突破**: 价格突破布林带后通常会回归中轨
4. **时间周期**: 可根据需要调整检测时间范围
