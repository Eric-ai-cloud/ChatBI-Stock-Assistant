# ChatBI股票助手 - ARIMA预测功能说明

## 📋 功能概述

stock_assistant-2.py 在 stock_assistant-1.py 的基础上，新增了 **ARIMA股票价格预测功能**，可以基于历史数据预测未来N天的股票价格走势。

---

## 🎯 新增功能

### arima_stock 工具

**功能描述：**
使用ARIMA(5,1,5)模型预测股票未来价格走势，基于过去一年的历史数据进行建模。

**参数说明：**
- `ts_code`（必填）：股票代码，如 '600519.SH'、'000858.SZ'
- `n`（可选）：预测天数，默认7天

**工作流程：**
1. 从本地SQLite数据库获取该股票过去一年的历史收盘价数据
2. 对时间序列数据进行预处理（重采样、向前填充缺失值）
3. 使用ARIMA(5,1,5)模型进行训练
4. 预测未来N天的收盘价及95%置信区间
5. 生成可视化图表（历史数据 + 预测结果）

**返回内容：**
- Markdown格式的预测结果表格
- 可视化图表（历史收盘价 + 预测价格 + 置信区间）

## 🔧 技术细节

### ARIMA模型参数

- **模型类型**: ARIMA(5,1,5)
  - p=5: 自回归阶数
  - d=1: 差分阶数（使序列平稳）
  - q=5: 移动平均阶数

### 数据处理

1. **数据获取**: 从SQLite数据库获取过去365天的历史数据
2. **时间序列处理**: 
   - 转换为DatetimeIndex
   - 按日频率重采样（asfreq('D')）
   - 向前填充缺失值（method='ffill'）
3. **模型训练**: 使用statsmodels库的ARIMA模型
4. **预测**: 获取未来N天的点预测和置信区间

### 可视化图表详解

#### 📊 综合对比图特性

ARIMA预测生成的折线图具有以下核心特性：

1. **数据纯净性**
   - ✅ **历史数据**：从SQLite数据库获取，天然仅包含交易日（无周末、节假日）
   - ✅ **预测数据**：通过智能过滤算法，仅生成未来N个**交易日**的预测
   - ✅ **无空值处理**：所有数据点均为有效交易日的收盘价

2. **连贯性保证**
   ```python
   # 提取历史最后一个交易日
   last_history_date = recent_history.index[-1]
   last_history_price = recent_history.values[-1]

   # 拼接历史终点和预测起点（均为交易日）
   connected_dates = [last_history_date] + list(forecast_dates)
   connected_prices = [last_history_price] + list(forecast_mean.values)

   # 绘制连贯的预测线（无断点）
   ax.plot(connected_dates, connected_prices, ...)
   ```

3. **视觉层次**

   | 元素 | 样式 | 说明 |
   |------|------|------|
   | 完整历史 | 浅蓝色细线 (alpha=0.7) | 过去一年所有交易日背景趋势 |
   | 近期历史 | 深蓝色粗线 + 圆点 | 最近60个交易日详细走势 |
   | 预测数据 | 紫色虚线 + 方块 | 未来N个交易日预测价格 |
   | 置信区间 | 紫色半透明阴影 | 95%不确定性范围 |
   | 分隔线 | 灰色虚线 | 区分历史和预测区域 |
   | "今天"标注 | 黄色框 | 当前日期定位 |

4. **X轴优化**
   - **数据类型**：pandas DatetimeIndex（纯交易日）
   - **刻度间隔**：每2周显示一个日期标签
   - **格式**：`YYYY-MM-DD`
   - **旋转**：45度，确保可读性

#### 🎯 为什么只绘制交易日？

1. **真实性**：股票市场仅在交易日开盘，非交易日无价格数据
2. **专业性**：符合金融分析标准，避免误导性展示
3. **连贯性**：排除非交易日后，折线图自然连贯，无断点
4. **准确性**：ARIMA模型基于交易日序列建模，预测结果也应对应交易日

#### ⚠️ 重要提示

- 预测的"N天"指的是N个**交易日**，而非自然日
- 如果跨越周末或节假日，实际覆盖的自然日会多于预测天数
- 例如：预测5个交易日，可能跨越7-10个自然日
- 当前节假日识别为近似估算，生产环境建议使用专业API

# ARIMA股票预测功能使用说明

## 功能概述

ARIMA股票预测工具 (`arima_stock`) 使用 ARIMA(5,1,5) 时间序列模型，基于过去一年的历史数据预测未来N天的股票收盘价。

## 技术规格

- **模型**: ARIMA(5,1,5)
- **数据源**: 本地SQLite数据库 `stock_prices.db`
- **历史数据范围**: 过去365天（一年）
- **预测参数**: 
  - `ts_code`: 股票代码（必填），如 `600519.SH`、`000858.SZ`
  - `n`: 预测天数（可选，默认7天）
- **交易日处理**: ✅ 智能识别交易日，自动排除周末和法定节假日

### 🆕 交易日识别策略

系统采用**多层次过滤策略**确保预测日期仅包含交易日：

#### 第一层：周末排除
- 自动跳过周六（weekday=5）和周日（weekday=6）

#### 第二层：固定假日排除（近似估算）
- **元旦**: 1月1日
- **劳动节**: 5月1日
- **国庆节**: 10月1日-3日
- **春节**: 2月初（简化处理，实际日期每年不同）

#### ⚠️ 重要说明
由于中国法定假日（特别是春节、清明、端午、中秋）的日期每年不同，且调休安排复杂，当前实现采用**近似估算**。如需更高精度，建议：

- Tushare Pro: `pro.trade_cal()`

## ❓ 常见问题与解决方案

### 问题1：预测折线图出现水平直线

**现象**：ARIMA预测结果的折线图中，预测部分呈现近乎水平的直线，价格变化极小。

**原因**：
- ❌ **错误做法**：对交易日数据进行了自然日频率重采样（`asfreq('D', method='ffill')`）
- 导致周末和节假日的价格被填充为前一个交易日的价格
- ARIMA模型学习到"价格经常保持不变"的错误模式
- 预测结果趋于平稳，失去预测价值

**解决方案**：
``python
# ✅ 正确做法：直接使用原始的交易日序列
ts_data = df_history['close_price'].copy()
# 不进行任何重采样，保持交易日的原始时间索引
```

**原理**：
- 股票市场仅在交易日开盘，周末和节假日本身就没有价格数据
- ARIMA模型应该基于**真实的交易日序列**进行训练
- 交易日的DatetimeIndex本身就是不规则的时间序列，ARIMA可以正确处理
- 预测时也只生成交易日的日期，保持数据的一致性

### 问题2：预测精度不高

**可能原因**：
1. **历史数据不足**：建议至少使用6个月以上的历史数据
2. **市场剧烈波动**：ARIMA适合平稳序列，剧烈波动时预测效果下降
3. **模型参数固定**：当前使用固定的ARIMA(5,1,5)，可尝试自动选择最优参数

**改进建议**：
- 增加历史数据量
- 结合其他技术指标（如MACD、KDJ）进行综合判断
- 生产环境可使用 `auto_arima` 自动选择最优参数

### 问题3：节假日识别不准确

**说明**：
- 当前实现仅能识别固定日期的假日（元旦、劳动节、国庆节）
- 春节、清明等农历节日采用近似估算，可能存在误差
- 调休安排复杂，无法精确处理

**生产环境建议**：
- 接入Tushare Pro API：`pro.trade_cal(exchange='SSE')`
- 或使用akshare库：`ak.tool_trade_date_hist_sina()`
- 获取官方发布的交易日历，确保准确性
