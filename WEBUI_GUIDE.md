## 🚀 快速启动

### Windows 用户

双击运行 `start_webui.bat` 文件，或在命令行执行：

```bash
start_webui.bat
```

### Linux/Mac 用户

```bash
# 添加执行权限
chmod +x start_webui.sh

# 运行
./start_webui.sh
```

### 直接运行

```bash
python webui.py
```

启动成功后，浏览器会自动打开 **http://localhost:7860**

## 💡 功能说明

### 1. 数据查询

**支持的查询类型：**

- 单只股票走势查询
  ```
  贵州茅台最近一个月的股价走势
  五粮液上周的交易数据
  ```

- 多股票对比分析
  ```
  对比五粮液和中芯国际2024年的涨跌幅
  比较贵州茅台和广发证券本月的表现
  ```

### 2. 预测分析

**ARIMA 股价预测：**

```
预测贵州茅台未来7天的股价
预测五粮液未来30天的走势
```

系统会基于过去一年的历史数据进行预测，并生成预测图表。

### 3. 技术分析

**布林带检测：**

```
检测贵州茅台的布林带超买超卖点
分析中芯国际的技术指标
```

系统会自动识别异常交易点，并在图表中标注。

### 4. 可视化展示

- 📈 自动生成交互式图表
- 📊 支持多种图表类型
- 🎨 清晰的视觉呈现
- 💾 图表自动保存到 `data/image_show/` 目录

## ❓ 常见问题

### Q1: 启动时提示 "未设置 DASHSCOPE_API_KEY"

**解决方案：**

1. 在启动脚本中输入 API Key
2. 或提前设置环境变量：

```bash
# Windows PowerShell
$env:DASHSCOPE_API_KEY="your-api-key"

# Linux/Mac
export DASHSCOPE_API_KEY="your-api-key"
```

### Q2: 无法访问 http://localhost:7860

**解决方案：**

1. 检查防火墙设置
2. 尝试其他端口：修改 webui.py 中的 server_port 参数
3. 查看控制台输出的实际地址

### Q3: 查询结果加载缓慢

**可能原因：**

- 网络延迟（调用 LLM API）
- 数据库查询复杂度高
- ARIMA 模型计算耗时

**优化建议：**

- 简化查询条件
- 缩短时间范围
- 检查网络连接

### Q4: 如何清空对话历史？

点击界面上的 "🗑️ 清空历史" 按钮即可。

### Q5: 支持哪些股票？

目前支持以下股票：

- 贵州茅台 (600519.SH)
- 五粮液 (000858.SZ)
- 广发证券 (000776.SZ)
- 中芯国际 (688981.SH)

## 🔧 高级配置

### 修改端口

编辑 `webui.py` 文件的 `main()` 函数：

```python
webui.run(
    server_port=8080,  # 修改此处
    share=False
)
```

### 启用公开分享

将 `share=False` 改为 `share=True`，Gradio 会生成一个临时公网链接。

### 自定义主题

在 `create_interface()` 函数中修改主题参数。
